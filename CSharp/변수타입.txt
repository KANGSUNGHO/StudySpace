특정이벤트에 대한 이벤트 핸들러에 대해 코딩
계산 버튼이 눌리면 계산하는 코드를 작성
윈도우 forms 어플리케이션은 이벤트가 발생했을 때
이벤트를 처리하는 핸들러가 있음.

* 문자형
int - 숫자를 저장하는 타입
string - 문자형
bool - true,false
어떤 타입의 데이터를 저장할 것인지 생각

int number = 10;
string operator1  = "+";
bool isCorrect = true;

int sum = 1 + 2;

int number1 = 1;
int number2 = 2;
int sum = number1 + number2;

HelloLabel.Text = sum.ToString(); // int 형인 sum을 문자열로 바꿔줌

모든 문장의 끝은 세미콜론(;)을 붙임
